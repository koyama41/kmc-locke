編著者後記

　ロックはこの世で最強のエスパーです。人間を越えた「超人」です。それなら、ロックの望みは全てかなえられるのかというと、決してそうではありません。むしろ、自分がいくら能力を使っても思うように事が進まずに苦悩し続けるのがロックの物語であると言えます。
　無限の力を持つ史上最強のエスパーであるにもかかわらず、世界を平和にする事ができないし、それどころか、自分の幸福すら長くは続かない、それがロックなんですねえ。
　これに対しては、人の能力はしょせん万能ではないとか、社会組織に対して個人は無力であるとか、 いろいろな見方ができるでしょうが、そうではなくて、どんなに苦しい思いをしながらも決してひねくれない、希望を捨てないそういう強い生き方を演出するために必要な設定だというのが本当のところだと思います。
　もっと利口に、器用に、上手に生きることができるはずなのに、そうしない、そうできないところがいかにも少年マンガのヒーローだなあと思いつつも、読み出すとやめられない、振り出すと止まらない（実はこれを言いたい！）ロックがとても好きです。

ＨＴＭＬ版編著者後記

　ＨＴＭＬ版ＫＭＣ超人ロックルールブックの作成は 1999.7.24 に始まりました。最初はTeX版ルールブック "Card Game LOCKE THE SUPERMAN Rule Book" (1995年3月3日発行, 106ページ) のソースファイルを LaTeX2HTML で変換して一丁上がりだろうと軽く考えていましたが、表や参照が gif になってしまうことや読みやすさの点で問題があり、一大奮起してドラフトから書き起こすことにしました。HTMLエディタとして Netscape Composer を使い、カット＆ペーストを駆使して 1999.8.1 には暫定版をネットにアップしましたが、URLと A NAME の文字列に２バイトコードを使ってしまったのが問題でした。幸い、小山氏の協力を得て、問題の２バイトコード文字を ASCII 文字列に変換するスクリプトによって文字コードの問題はほとんど解決することができました。A NAME タグに出てくる意味不明の文字列はその時の自動変換の名残です。

　このHTML化作業と並行する形で、TeX 版ルールブックの内容に関する議論がLocke-MLにおいて進められました。その結果、20箇所以上にわたって記述を改め、説明を追加し、内容的にかなり充実したものにすることができました。改訂版ルールブックは 1999.8.17 にネットにアップされ、編集作業は一段落しました。

　その後は、2001.3.7 のホームページの移動、2002.1.6 の超人ロック戦略ブック作成の影で少しずつ整備作業を進めてきました。今回、ようやく索引の全項目がそろったことで、大きな節目を迎えたものと考えています。HTML版ルールブックは、2002.06.10修正版をもって「2001年版ルールブック」として内容をある程度固定し、以後の大きな変更は別バージョンとして整備したいと思います。その際は、できれば章立てから見直し、超人ロックそのものを知らない方にも分かりやすく楽しめる内容にしていきたいと思います。

では、今後ともＫＭＣ超人ロックルールブックをよろしくお願いいたします。

(2002.06.10 よこちょん)
